Lineage  ▼►

Lineage statement 
Source data for this project includes the NSW Catchment Management Authority boundaries and the IBRA Sub-region V.6  boundaries. These two layers were unioned to create a new layer. Linework from the Catchment Management Authority (CMA) dataset was given priority in areas where linework was duplicated. For example the NSW state border which forms the boundary to the study area was taken from the CMA layer. 

The IBRA Sub-region v.6 Unit titled South East Coastal Ranges was split in two creating a new unit called South East Coastal Plains. The polygon was split using a contour line. 

The resulting layer was cleaned by removing slivers and dissolving polygons less than 1,000 hectares with the adjoining IBRA polygon within the same CMA area. Polygons between 1,000 and 20,000 hectares were also selected for review. If a polygon within this size class was found to be the only representative of a particular IBRA Sub-regional type within a CMA area they were merged with the most logical adjoining polygon. Polygons between 1,000 and 20,000 ha were retained where there were other representive samples of a particular IBRA Sub-region type within a CMA.

Six (6) polygons present in Version 1 of the data were absorbed into larger adjoining polygons in Version 2. This was done to reduce the number of subdivisions within a CMA.

Updated November 2012.
